== Description ==

### Shortcode To Widget 

Display any shortcode to your sidebar easily,
for example display [Perso Slider](http://wp-time.com/perso-slider/) or [Facebook Post Embed](http://wp-time.com/facebook-post-embed/) or [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) or any shortcode to your sidebar.

### The Features

* Display any shortcodes to sidebar.
* Easy to use.

== Installation ==

### Installation

1. Upload 'shortcode-to-widget' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets > Shortcode To Widget.
4. Drag and drop "Shortcode To Widget" widget and open it and enter your shortcode.
